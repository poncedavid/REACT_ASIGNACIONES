# Coding Dojo - Bootcamp MERN
## BID-api-bromas
### Tarea core
#### Descripción
Desarrollo de una API de bromas, con conexión a una base de datos Mongoose local, express js y dotenv
[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://choosealicense.com/licenses/mit/)
#### Instalación
- Clonar o descargar el repositorio
- Instalar las dependencias:
    1. Utilizando npm
    ```bash
    npm install
    ```

    2. Utilizando yarn
    ```bash
    yarn
    ```

    3. Utilizando pnpm
    ```bash
    pnpm install
    ```

- Ejecutar el proyecto:
    1. Utilizando npm
    ```bash
    npm run dev
    ```

    2. Utilizando yarn
    ```bash
    yarn dev
    ```

    3. Utilizando pnpm
    ```bash
    pnpm dev
    ```
- Habilitar el servicio mongod para la conexión y creación de una base de datos local de mongodb
- Configuración del archivo .env
- Servidor local: http://localhost:4000
